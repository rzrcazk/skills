### 第3幕：定义 - 定理表述
**画面**: 公式 "a² + b² = c²" 动画显示 + 直角三角形标注
**布局**:
- 公式在底部公式区 (y: -4.5)

**字幕**: 直角三角形中，a² + b² = c²
**读白**: "勾股定理说：在直角三角形中，两条直角边的平方和，等于斜边的平方。注意哦，这个定理只适用于直角三角形！"
**动画**:
- 0.0s: 三角形淡入
- 1.0s: 逐边标注a、b、c
- 2.0s: 直角标记闪烁高亮（红色）
- 3.0s: 公式逐字显示
- 5.0s: "仅适用于直角三角形"文字高亮
**目的**: 明确定理内容 + 适用条件
**时长**: 10秒

### 第4幕：定义 - 图形验证
**画面**: 边长为3、4、5的直角三角形，分别在三条边上画正方形
**布局**:
- 三角形居中偏上 (y: 1.0)
- 三个正方形从边向外生长
- 面积标注在正方形中央
**字幕**: 9 + 16 = 25 ✓
**读白**: "看这个3-4-5三角形，直角边上的两个正方形面积加起来，正好等于斜边上正方形的面积！"
**动画**:
- 0.0s: 高亮直角边上的两个正方形（面积9和16）
- 2.0s: 高亮斜边上的正方形（面积25）
- 4.0s: 显示等式 9 + 16 = 25
- 6.0s: 打勾确认
**目的**: 直观展示定理（不解释3²=9，直接给面积）
**时长**: 10秒

### 第5幕：探究 - 证明动画（面积法）
**画面**: 赵爽弦图 - 逐步展示证明过程
**布局**:
- 大正方形居中 (y: 0.5)
- 面积等式在底部 (y: -4.5)
- 字幕在字幕区 (y: -3.5)
**字幕**: 面积法证明
**读白**: "古代中国人给出了漂亮的证明！来看这个图，大正方形边长是a+b，里面放了四个相同的直角三角形..."

**动画分镜**：
- 0.0s: **绘制大正方形**，标注边长 (a+b)
- 2.0s: **放入四个三角形**，用不同颜色区分，标注直角边 a, b 和斜边 c
- 5.0s: **揭示内部小正方形**，边长正好是 c
- 8.0s: **计算大正方形面积**: (a+b)² = a² + 2ab + b²
- 11.0s: **另一种算法**: 四个三角形面积 + 小正方形面积 = 4×(½ab) + c² = 2ab + c²
- 14.0s: **等式对比**: a² + 2ab + b² = 2ab + c²
- 16.0s: **消去 2ab**，得出 **a² + b² = c²**（高亮显示结论）

**目的**: 完整展示证明过程，每步有对应图形
**时长**: 18秒

## 备注

1. **5段式结构**：引入→定义→探究→拓展→总结，每段有明确检查项
2. **知识水平**：初中 - 禁止展示 3²=9 等基础计算，重点在证明过程

   - 面积法必须分步骤展示（大正方形→三角形→小正方形→等式推导）
   - 每步动画对应一句读白
   - 最终结论必须高亮
7. **实例要求**：
   - 生活例子必须有示意图
   - 示意图包含：场景轮廓 + 关键尺寸标注 + 计算过程
8. **风格**：幽默生动，用日常生活例子
9. **系列感**：固定开场结尾格式 + 5段式统一结构
10. **音频情感**：热情 → 活泼 → 自豪 → 温和 → 惊叹 → 神秘 → 亲切 → 提醒 → 温暖 → 不舍
